# TableData
Use this project if you need to display data in table format.

## Features
* HTML conformant: it uses THEAD, TH, TBODY, TR, TD
* Column ordering
* Column size
* Cell filter: a filter function can be used to change data before displaying it
* Built-in filters:
  * Check
  * CheckAll
  * Status
